## `test/feature/`

Unit tests for the `feature/` layer, mirrored by subpackage:

- `ingestion/`
- `enrichment/`
- `lineage/`
- `extraction/`
- `ownership/`
- `versioning/`
- `profiling/`
- `rbac/`

All tests are designed to run **offline** (no real DataHub/network) using mocks/stubs.
